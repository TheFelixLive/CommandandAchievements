## About
Command & Achievement is a comprehensive command utility addon for Minecraft Bedrock, designed to enhance your command execution experience. While fully compatible with hardcore mode, it serves as a powerful framework for managing and automating commands, complete with visual interfaces, history tracking, command chains, and permission controls.

This addon transforms the way you interact with Minecraft commands, making them accessible, repeatable, and secure in multiplayer environments.

## Features

### Command Chains
Create sequences of commands that run automatically. Use the UI editor to build chains, chose an icon, pin them to the main menu, and execute them with a single click or in the chat. Automate complex tasks, set up triggers, and build reusable command workflows. A powerful feature for advanced users and server administrators.

### Visual Commands
Execute commands through an intuitive graphical interface instead of typing them manually. Configure common commands visually for quick access and error-free execution.

![Minecraft 24 06 2025 22_07_41](https://github.com/user-attachments/assets/4baef785-b802-432f-91d6-f507472f21d5)

### Command History
Every executed command is saved to your world, allowing you to rerun them anytime. Perfect for console players and anyone who wants to maintain a personal command library.

![Minecraft 23 06 2025 17_11_28](https://github.com/user-attachments/assets/79047fdf-9972-4bc0-a593-55314e98072b)

### Player-Specific Permissions
Grant command execution permissions to specific players in multiplayer worlds. This feature allows you to control who can access certain commands, controling there settings or disabling sertain features for them such as the main menu or the ability to use chain commands.
Only the world admins have access to commands, but you can easily manage permissions through Minecraft's built-in player management system or grant those permissions as a Host player.

### Error Handling
Get clear feedback when commands fail due to syntax errors or Minecraft limitations. The addon displays helpful error messages to guide you.

## More Highlights
### Achievements Compatibility
This addon is fully compatible with Minecraft's achievement system and does not disable the achievements. You can continue to earn achievements while using the addon.

### Custom Command Support
Most commands can simply executed by typing them in the `com2hard:` prefix in the chat. This allows you to use the addon’s features without relying solely on the visual interface, giving you flexibility in how you interact with commands.
It even extends to menu shortcuts like:
- `/com2hard:menu`
- `/com2hard:chains`
- `/com2hard:history`
- `/com2hard:settings`

### Added Support for /agent Commands
The addon now includes support for executing `/agent` commands, allowing you to manage and control agents directly through the visual command interface. This integration enhances your ability to utilize agents in your Minecraft world, making it easier to execute complex tasks and automate processes involving agents. For players who allready used agents, this addon moddefies slightly the way they work, to be able to remove him from the world.

### Extensive Logging System for Admins
Admins can access a comprehensive logging system that records all command executions, including who executed them and when. This feature is invaluable for monitoring command usage and maintaining control over your multiplayer environment.

### Hardcore Mode Compatibility
While this addon provides extensive command utilities, it includes full compatibility with hardcore mode, allowing you to use these features even in the most challenging gameplay conditions.

### Location Insertion
Easily insert your current location into commands. Just click the `Location Insertion` button using the visual command interface and you can look at a location to insert its coordinates. This feature simplifies the process of executing location-based commands like `/tp` or `/fill`, making it more convenient to use commands that require coordinates.

## How to Use
1. Open the menu (/menu or a shortcut).
2. Navigate through the dashboard to access Visual Commands, History, Chains, Permissions, or Settings.
3. Select and execute commands as needed.

## FAQ & Support
Q: Is this addon compatible with multiplayer servers?
A: Yes, it is designed to work in multiplayer environments, allowing you to manage permissions and execute commands with ease.

Q: Can I use this addon in hardcore mode?
A: Yes, the addon is fully compatible with hardcore mode, allowing you to use its features without any restrictions.



## Installation
1. **Download** the latest release from [here](https://github.com/TheFelixLive/Command2Hardcore/releases/latest).
2. **Open** the `.mcpack` or `.mcaddon` file with Minecraft.
3. **Create** a new world OR **edit** an existing world.
4. **Navigate to**: Behavior packs.
5. **Click on** "Available".
6. **Activate** the Addon.
7. `Play` or `Create` your World.
